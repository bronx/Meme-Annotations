Instruções:


Anote seu código. Só isso.

Exemplos:


@FuckYea:

@FuckYea
public void algumMetodo() {
// Resolve algo complexo
}


@Ffffuuuuuuu:

@Ffffuuuuuuu

// Deveria estar funcionando (mas não está)
}


@Trollface:

@Trollface
public boolean isAlgo() {
return this.equals(this);
}

@MeGusta
public List<Algo> algumMetodo() {

}

@HatersGonnaHate

}

@ForeverAlone
public class MySingleton {

}


@Problem:

@Problem("?")
public void algumMetodo() {
// Não deveria estar funcionando, mas está!
}
